Gemma 4 API 설정 방법
  1. Google AI Studio 또는 Google Cloud에서 API 키를 발급합니다.
  2. Vercel 프로젝트의 Settings → Environment Variables에 GEMINI_API_KEY를 추가합니다.
  3. 필요하면 모델명을 GEMMA_MODEL 환경변수로 바꿀 수 있습니다. 기본값은 gemma-4-26b-a4b-it입니다.
  4. 다시 배포하면 Gemma 4 분석이 작동합니다.
